# 기능 목록

- 사람이 1명이고, 높이가 1이라면:
  - 1 => 1

- 사람: 2, 높이: 1
  - 발판이 1개일 경우:
    - 1 => 2
    - 2 => 1

- 사람: 2, 높이: 2
  - 발판이 2개일 경우:
    - 1 => 1
    - 2 => 2

- 사람: 3, 높이: 1
  - 왼쪽에 발판이 1개일 경우:
    - 1 => 2
    - 2 => 1
    - 3 => 3
  - 오른쪽에 발판이 1개일 경우:
    - 1 => 1
    - 2 => 3
    - 3 => 2

- 사람: 3, 높이: 2
  - 높이 1에서 왼쪽에 발판이 1개, 높이 2에서 오른쪽에 발판이 1개:
    - 1 => 3
    - 2 => 3
    - 3 => 2
